Rounding

round() fix() floor() ceil
四捨五入 取整數位 捨去 進位 












A = [2:0.4:2];
A = 2 1.6 1.2 0.8 0.4 0 0.4 0.8 1.2 1.6 2

round(A) =
        -2 2 1 1 0 0 0 1 1 2 2
fix(A)= //towards zero
        -2 1 1 0 0 0 0 0 1 1 2
floor(A) = 
        -2 2 2 1 1 0 0 0 1 1 2
ceil(A) = 
        -2 1 1 0 0 0 1 1 2 2 2

Data analysis

  • max() sum() size() min()

  • prod(): product all elements in matrix A (把每個column都算出一個乘積)

  • length(): 高度

  • mean(v): 預設1-dimension

  • mean(v,2): 輸出為2-dimension

  • sort(): sort every column

  • std(): 標準差

  • var()

  • median(): 中位數

  • sortrows(A): sort rows of matrix A by column 1 (小到大)

  • sortrows(A, 2): sort rows of matrix A by column 2 (小到大)

  • sortrows(A, -2): sort rows of matrix A by column 2 (大到小)

  • sortrows(A, [-2 3]): sort rows of matrix A by column 2 (大到小) 。如果值相同，用第三個column排序(小到大)

Generation of random numbers

Normal random numbers













rng = ('default')     // for reproducibility
r = normrnd(0,1)     // Save the current state of the random number generator.

// 把現在的random狀態儲存在s
s = rng
r = normrnd(0,1)     // Restore the state of the random number generator to s

// 把之前的random state拿出來用設為現在的state
rng(s);
r = normrnd (0,1)    // same set of numbers are generated

Standard normal random numbers

  • create number
